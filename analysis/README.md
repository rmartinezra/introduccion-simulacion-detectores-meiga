# Análisis

Aquí se construirá un paquete Python único para leer, validar, resumir y visualizar resultados MEIGA.

La primera versión deberá ofrecer:

- lectura robusta de JSON/JSON comprimido;
- resumen por detector y partícula;
- distribuciones de energía depositada, carga y tiempo;
- comparación de dos o más simulaciones;
- exportación de tablas y figuras;
- interfaz de línea de comandos y API importable;
- pruebas con datos pequeños y deterministas.

Los scripts históricos de análisis se usarán como referencia, no se copiarán sin revisión.
